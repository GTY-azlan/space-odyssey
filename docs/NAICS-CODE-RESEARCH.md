## 1. Executive Summary

This document presents a comprehensive, PhD-level analysis of all North American Industry Classification System (NAICS) codes applicable to **Space Odyssey** — a web development studio and enterprise software platform builder that creates:

1. **Custom websites and web applications** for clients (service-based revenue)
2. **Software platforms and SaaS products** (product-based revenue)
3. **Interactive digital experiences** — data dashboards, research platforms, astronomy portals
4. **Enterprise-grade multi-tenant platforms** with SSO, analytics, and white-label capabilities

After exhaustive research of the official NAICS 2022 taxonomy, U.S. Census Bureau cross-references, and SBA Size Standards (effective March 17, 2023), this analysis identifies **6 directly applicable codes** and **4 adjacent codes**, with detailed justification for each inclusion and exclusion.

### Key Finding

The Space Odyssey platform operates at the **intersection of three distinct business models**, each mapping to a different NAICS code:

| Business Model | Primary NAICS Code | SBA Size Standard |
|---|---|---|
| Custom web/app development for clients | **541511** | $32.5M annual receipts |
| Platform/SaaS product publishing | **511210** | $41.5M annual receipts |
| Integrated systems & architecture design | **541512** | $32.5M annual receipts |

**The strategic recommendation is to register under 541511 as the primary code, with 511210 and 541512 as secondary codes** — this maximizes eligibility for federal contracts, SBA programs, and small business certifications.

### Analytical Framework

Each NAICS code was evaluated against the following criteria:

1. **Primary Revenue Match** — Does the official industry definition align with Space Odyssey's core revenue-generating activities?
2. **Activity Overlap** — Do the included activities listed in the NAICS manual match what Space Odyssey actually does?
3. **Exclusion Analysis** — Are there explicit exclusions that would disqualify Space Odyssey from this code?
4. **Cross-Reference Consistency** — Do cross-references to other codes create conflicts or reinforce the classification?
5. **Strategic Value** — What government contracting, SBA loan, or certification advantages does this code provide?
6. **Size Standard Generosity** — Is the SBA size standard (in $M receipts) favorable for the projected revenue trajectory?

Before mapping NAICS codes, we must rigorously deconstruct what Space Odyssey **actually does** as a business entity. This deconstruction is critical because NAICS classification follows **revenue-generating activities**, not technological capabilities.

### 3.2 Activity-to-Industry Mapping

| Space Odyssey Activity | Industry Category | Rationale |
|---|---|---|
| Building custom websites for clients | Custom Programming | Writing code to meet specific customer needs |
| Designing platform architecture | Systems Design | Planning integrated hardware/software/communication systems |

Understanding the revenue breakdown is therefore essential for strategic code selection.

## 4. Primary NAICS Codes — Detailed Analysis

### 4.1 NAICS 541511 — Custom Computer Programming Services

#### Official Definition

> *"This U.S. industry comprises establishments primarily engaged in writing, modifying, testing, and supporting software to meet the needs of a particular customer."*

#### SBA Size Standard: **$32.5 Million** (Average Annual Receipts)

#### Sector Classification
- **Sector:** 54 — Professional, Scientific, and Technical Services
- **Subsector:** 5415 — Computer Systems Design and Related Services
- **Industry Group:** 54151 — Computer Systems Design and Related Services

#### Included Activities (Official NAICS Index)

| Activity | Relevance to Space Odyssey |
|---|---|
| Applications software programming services, custom computer | ✅ Directly applicable — building web apps for clients |
| Computer program or software development, custom | ✅ Directly applicable — custom platform development |
| Computer programming services, custom | ✅ Directly applicable — core business activity |
| Computer software analysis and design services, custom | ✅ Directly applicable — architecture consulting |
| Computer software programming services, custom | ✅ Directly applicable — all development work |
| Computer software support services, custom | ✅ Directly applicable — platform maintenance |
| Web (i.e., Internet) page design services, custom | ✅ **Directly applicable** — website creation |
| Software analysis and design services, custom computer | ✅ Directly applicable — platform architecture |

#### Excluded Activities & Cross-References

| Exclusion | Cross-Reference | Impact on Space Odyssey |
|---|---|---|
| Software design, development, and publishing, or software publishing only | **511210** (Software Publishers) | If Space Odyssey's PRIMARY revenue is from selling its own platform (not client work), this code takes precedence |
| Planning and designing computer systems that integrate hardware, software, and communication technologies | **541512** (Computer Systems Design) | If Space Odyssey provides integrated systems design (not just code), 541512 may be more appropriate |

#### Why This Code Applies — Detailed Justification

**NAICS 541511 is the single most applicable code for Space Odyssey** for the following reasons:

1. **Explicit Web Design Inclusion:** The official NAICS index explicitly lists *"Web (i.e., Internet) page design services, custom"* as an included activity. This is a direct, unambiguous match to Space Odyssey's core service of building custom websites and web applications for clients.

2. **Custom Software Mandate:** Every project Space Odyssey undertakes involves writing, modifying, testing, and supporting software to meet the specific needs of a particular customer (client or platform user). This is the exact definition of the industry.

3. **Broad Federal Contracting Usage:** NAICS 541511 is one of the most frequently used codes in federal IT contracting. In FY2024, over **$6.3 billion** in federal contracts were awarded under this code alone, spanning agencies from NASA to DHS.

4. **Size Standard Advantage:** At $32.5 million in annual receipts, a small web development studio can remain classified as a "small business" for federal contracting purposes well into the seven-figure revenue range.

5. **No Capital Requirement:** Unlike manufacturing or publishing codes, 541511 has no minimum capital, facility, or employee requirements — it is purely skill-based.

#### Why This Code Might NOT Be the Best Fit

**Scenario:** If Space Odyssey transitions from a service business (building for clients) to a product business (selling its own platform/SaaS), then **511210 (Software Publishers)** becomes the more appropriate primary code, because 541511 explicitly excludes establishments *"primarily engaged in software design, development, and publishing."*

**Mitigation:** As long as Space Odyssey generates a significant portion of revenue from client-facing custom development work, 541511 remains valid even alongside product revenue.

#### Companies Classified Under 541511

Notable companies classified under this code include: **Cognizant Technology Solutions, EPAM Systems, Workday Inc., Zoom Video Communications, and Take-Two Interactive Software.**

### 4.2 NAICS 541512 — Computer Systems Design Services

#### Official Definition

> *"This U.S. industry comprises establishments primarily engaged in planning and designing computer systems that integrate computer hardware, software, and communication technologies. The hardware and software components may be provided directly by the establishment as part of integrated services or sourced from third-party vendors. Establishments in this industry typically also install the systems and provide user training and ongoing support."*

#### SBA Size Standard: **$32.5 Million** (Average Annual Receipts)

#### Sector Classification
- **Sector:** 54 — Professional, Scientific, and Technical Services
- **Subsector:** 5415 — Computer Systems Design and Related Services

#### Included Activities

| Activity | Relevance to Space Odyssey |
|---|---|
| Computer systems integration design consulting | ✅ Directly applicable — platform architecture consulting |
| Local area network (LAN) and network systems integration design | ⚠️ Partially applicable — API integrations, not physical LANs |
| Information management and office automation systems integration design | ✅ Directly applicable — data dashboard platforms |
| Computer-aided design (CAD), engineering (CAE), and manufacturing (CAM) systems integration | ⚠️ Not applicable — Space Odyssey doesn't do CAD/CAM |
| Computer hardware and software consulting services | ✅ Partially applicable — software architecture consulting |

#### Excluded Activities & Cross-References

| Exclusion | Cross-Reference | Impact |
|---|---|---|
| Selling computer hardware/software from retail-like locations | **443142** (Electronics Stores) | Not applicable — Space Odyssey has no retail presence |
| Merchant wholesaling computer hardware/software | **423430** (Computer Equipment Wholesalers) | Not applicable — no wholesale activity |

#### Why This Code Applies — Detailed Justification

**NAICS 541512 applies to Space Odyssey when the business operates as a systems integrator and platform architect**, rather than merely a code writer. The key distinction from 541511 is:

- **541511** = Writing code to a specification (programming)
- **541512** = Planning and designing the entire system architecture (systems design)

Space Odyssey's RESEARCH.md document outlines plans for:
- Real-time data integration (NASA APIs, NOAA space weather, SpaceX API)
- Multi-tenant architecture with SSO (SAML/OAuth)
- Role-based access control and audit logging
- Custom branding and white-label capabilities
- API documentation and developer tools
- Data export (CSV, JSON, PDF) and RSS feeds

These are **systems integration and design activities** — not just programming. When Space Odyssey advises a university on how to architect an enterprise astronomy platform that integrates external APIs, manages authentication, and serves multiple tenants, it is performing **computer systems design services**.

#### Why This Code Might NOT Be the Best Primary

**Limitation:** NAICS 541512 traditionally emphasizes **hardware-software integration** — planning systems that include physical infrastructure (servers, networks, communication equipment). While the definition includes software, the industry's historical focus on infrastructure integration means:

1. Federal contracts under 541512 tend to favor **systems integrators** (e.g., Accenture, Deloitte) over pure-play software developers.
2. The code is less specific to web development than 541511, which explicitly includes "web page design services."

**Strategic Use:** 541512 is best used as a **secondary code** when Space Odyssey provides end-to-end platform consulting that includes infrastructure planning, not just development.

### 4.3 NAICS 511210 — Software Publishers

#### Official Definition

> *"This industry comprises establishments primarily engaged in computer software publishing or publishing and reproduction. Establishments in this industry carry out operations necessary for producing and distributing computer software, including software design, development, and publishing."*

#### SBA Size Standard: **$41.5 Million** (Average Annual Receipts)

#### Sector Classification
- **Sector:** 51 — Information
- **Subsector:** 511 — Publishing Industries (except Internet)
- **Industry Group:** 5112 — Software Publishers

#### Included Activities

| Activity | Relevance to Space Odyssey |
|---|---|
| Software design and development | ✅ Directly applicable — platform development |
| Software publishing (licensing, distribution) | ✅ Directly applicable — SaaS platform licensing |
| Reproduction of software products | ✅ Applicable — multi-tenant platform deployment |
| Operating software support services | ✅ Directly applicable — platform maintenance & updates |

#### Excluded Activities & Cross-References

| Exclusion | Cross-Reference | Impact |
|---|---|---|
| Custom software programming for specific customers | **541511** (Custom Computer Programming) | If the PRIMARY activity is client-specific work, not product publishing |
| Publishing software exclusively on the Internet | **519130** (Internet Publishing) | If the PRIMARY activity is content publishing, not software distribution |

#### Why This Code Applies — Detailed Justification

**NAICS 511210 becomes the most appropriate primary code when Space Odyssey operates as a software product company** — selling licenses to its platform, offering SaaS subscriptions, or distributing reusable software products.

The critical distinction from 541511 is the **revenue model**:

| Aspect | 541511 (Custom Programming) | 511210 (Software Publishers) |
|---|---|---|
| Revenue Source | Client fees for development work | Licensing fees, subscriptions, product sales |
| Output | Custom software for one client | Standardized software sold to many |
| Relationship | Service provider → client | Publisher → licensee |
| Delivery | Project-based engagement | Product-based engagement |

Space Odyssey's RESEARCH.md outlines a vision for an **enterprise astronomy research platform** that would be:
- Licensed to university astronomy departments
- White-labeled for research organizations
- Sold as a SaaS subscription with tiered pricing
- Distributed with API access and developer tools

This is **software publishing** — creating a product once and selling it to many customers. This is fundamentally different from custom programming, where each engagement produces a unique deliverable for a specific client.

#### Why This Code Has the Highest Size Standard

At **$41.5 million**, NAICS 511210 has a higher SBA size standard than 541511 ($32.5M). This reflects the economics of software publishing:

1. **Scalability:** Software publishers can generate significant revenue from a single product through volume licensing.
2. **R&D Investment:** Publishing requires upfront investment in design, development, testing, and documentation before any revenue is generated.
3. **Distribution Costs:** Publishers bear the cost of marketing, sales, support, and updates across their entire customer base.

The higher threshold acknowledges that software publishing companies need more revenue to achieve profitability than service-based programming firms.

#### Strategic Implication

**If Space Odyssey's long-term vision is product-led growth** (selling platforms, not hours), then 511210 should eventually become the primary NAICS code, as it:
- Offers a higher size standard ($41.5M vs $32.5M)
- Better reflects the product-based revenue model
- Aligns with how federal agencies classify software product companies

### 4.4 NAICS 541519 — Other Computer Related Services

#### Official Definition

> *"This U.S. industry comprises establishments primarily engaged in providing computer-related services, explicitly excluding custom programming, systems integration design, and facilities management services."*

#### SBA Size Standard: **$32.5 Million** (Average Annual Receipts)

#### Sector Classification
- **Sector:** 54 — Professional, Scientific, and Technical Services
- **Subsector:** 5415 — Computer Systems Design and Related Services

#### Included Activities

| Activity | Relevance to Space Odyssey |
|---|---|
| Computer disaster recovery services | ⚠️ Not directly applicable |
| Computer software installation services | ✅ Applicable — deploying platforms for clients |

#### Excluded Activities & Cross-References

| Exclusion | Cross-Reference |
|---|---|
| Custom computer programming services | **541511** |
| Computer systems integration design services | **541512** |
| Computer facilities management services | **541513** |

#### Why This Code Applies — Limited Justification

**NAICS 541519 is a residual/catch-all code** for computer-related services that don't fit into the more specific categories. It has very limited applicability to Space Odyssey because:

1. **Narrow Scope:** The only two explicitly included activities are disaster recovery and software installation — neither is a core Space Odyssey activity.
2. **Exclusion of Core Activities:** The code explicitly excludes custom programming (541511), systems design (541512), and facilities management (541513) — which are Space Odyssey's primary activities.

**When It Might Apply:** If Space Odyssey offers services like:
- Platform migration services (moving clients from old systems)
- Software installation and configuration as a standalone service
- Data conversion or system upgrade services

**Recommendation:** Use as a **tertiary or supplementary code** only if Space Odyssey offers standalone services that don't fall under programming, systems design, or publishing.

### 4.5 NAICS 518210 — Data Processing, Hosting, and Related Services

#### Official Definition

> *"This industry comprises establishments primarily engaged in providing infrastructure for hosting or data processing services. Establishments may provide specialized hosting (e.g., web hosting, streaming services, application hosting, excluding software publishing) or general time-share mainframe facilities to clients."*

#### SBA Size Standard: **$32.5 Million** (Average Annual Receipts)

#### Sector Classification
- **Sector:** 51 — Information
- **Subsector:** 518 — Data Processing, Hosting, and Related Services

#### Included Activities

| Activity | Relevance to Space Odyssey |
|---|---|
| Application hosting | ✅ Directly applicable — hosting platform instances |
| Web hosting | ✅ Directly applicable — hosting client websites |
| Computer data storage services | ✅ Applicable — platform data management |
| Automated/electronic data processing | ⚠️ Partially applicable — platform data processing |
| Video, audio, and media streaming services | ⚠️ Potentially applicable — educational video content |

#### Excluded Activities & Cross-References

| Exclusion | Cross-Reference | Impact |
|---|---|---|
| Software design, development, & publishing | **511210** (Software Publishers) | Hosting alone doesn't make you a publisher |
| On-site management of client data processing facilities | **541513** (Computer Facilities Management) | Different from cloud-based hosting |
| Operating web search portals | **519130** (Internet Publishing) | If search is the primary function |
| Wired broadband Internet access + web hosting | **517311** (Wired Telecommunications) | Not applicable — no ISP activity |

#### Why This Code Applies — Conditional Justification

**NAICS 518210 applies to Space Odyssey when the business provides hosting infrastructure as a service**, not just software development.

Space Odyssey's platform architecture includes:
- Multi-tenant hosting on Vercel/cloud infrastructure
- Application hosting for white-label platform instances
- Data storage and processing for research databases
- Edge caching and CDN management

However, the **critical distinction** is whether hosting is a **primary revenue stream** or an **enabling function** for the software product.

**If Space Odyssey charges clients specifically for hosting** (e.g., "$500/month for platform hosting"), then 518210 is applicable as a revenue-generating activity.

**If hosting is bundled into the software license** (e.g., "$50,000/year platform license includes hosting"), then 511210 (Software Publishers) subsumes the hosting activity, and 518210 is not needed.

**Recommendation:** Use as a **secondary code** only if Space Odyssey offers standalone hosting services separate from its software products.

2. **Astronaut Database:** Searchable database of 350+ astronauts with career statistics and timelines.

These are **Internet-exclusive publishing activities** — the exact definition of NAICS 519130.

#### Why This Code Has a Lower Size Standard

At **$25.5 million**, NAICS 519130 has a lower size standard than 511210 ($41.5M) and the 54151x codes ($32.5M). This reflects the economics of Internet publishing:

1. **Lower Capital Requirements:** Internet publishing requires less upfront investment than software development.
2. **Revenue Model:** Content publishing typically generates lower per-unit revenue than software licensing.
3. **Market Dynamics:** The Internet publishing market is more competitive with lower barriers to entry.

**Recommendation:** Use as a **supplementary code** for the content publishing aspects of the platform, but not as a primary code.

#### Official Definition

> *"This U.S. industry comprises establishments primarily engaged in providing on-site management and operation of clients' computer systems and/or data processing facilities."*

#### Why This Code Does NOT Apply — Infrastructure Justification

NAICS 541513 is specifically about **on-site, physical facilities management** — managing the computer rooms, data centers, and physical infrastructure that belong to clients.

The distinction between 541513 and 518210 is critical:
- **541513** = Managing the client's physical data center (on-site)
- **518210** = Hosting the client's application in your own data center (cloud)

**Recommendation:** **Do not use** this code for Space Odyssey.

## 6. Excluded NAICS Codes — Why They Don't Apply

### 6.1 NAICS 541618 — Other Management Consulting Services

**Definition:** Establishments providing specialized management consulting services not classified elsewhere.

**Why Excluded:** While Space Odyssey provides platform strategy advice, it is not primarily a management consulting firm. The consulting is ancillary to the software development and publishing activities. Management consulting implies ongoing advisory relationships separate from product delivery.

### 6.2 NAICS 541690 — Other Scientific and Technical Consulting Services

**Definition:** Establishments providing advice on scientific and technical issues (except environmental).

**Why Excluded:** Although the Space Odyssey platform presents scientific content (astronomy data, NASA missions), Space Odyssey is not providing **scientific consulting** to clients. It is building platforms that present scientific data — a fundamentally different activity.

### 6.3 NAICS 541820 — Public Relations Agencies

**Definition:** Establishments developing and implementing public relations plans.

**Why Excluded:** Space Odyssey does not provide public relations services, media relations, or crisis communication management.

### 6.4 NAICS 541870 / 541890 — Advertising Material Distribution / Other Advertising Services

**Why Excluded:** These codes relate to physical advertising material distribution and miscellaneous advertising services — not applicable to a software/web development business.

### 6.5 NAICS 611420 — Computer Training

**Definition:** Establishments offering computer training (except vocational).

**Why Excluded:** While Space Odyssey's educational resources include quizzes and learning paths, training is a supporting feature of the platform, not a primary business activity.

All listed NAICS codes qualify for SBA 7(a) loans as long as the business remains under the applicable size standard. The size standard determines the maximum revenue before losing eligibility.

All listed codes qualify for SBA 504 loans (fixed asset financing), but the primary code affects the loan processing timeline:
- **541511:** Standard processing (service business — lower capital requirements)
- **511210:** May require additional documentation (product business — R&D asset valuation)

When registering on SAM.gov (System for Award Management), Space Odyssey should:
1. List **541511** as the primary NAICS code

3. Update the primary code if the revenue model shifts from services to products

### 10.4 DUNS/UEI Implications

The NAICS code on file determines which federal opportunities appear in the entity's SAM.gov dashboard. Using the right codes ensures the business receives relevant solicitation notifications.
